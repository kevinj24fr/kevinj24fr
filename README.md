## Mission

I lead an interdisciplinary research program where **single-cell & spatial transcriptomics**, **AI**, and **neurotechnology** converge. We aim to transform **high-dimensional biological data** into **clinical insights** and to advance fundamental understanding of how the brain responds to tumors, trauma, and intervention.

## Rust Tools for Bioinformatics

Fast, single-binary CLI tools for common bioinformatics tasks. No Python startup time, no dependency hell.

| Tool | What it does |
|------|-------------|
| [**h5utils**](https://github.com/kevinj24fr/h5utils) | Operate on h5ad (AnnData) files without full materialization |
| [**peek-series**](https://github.com/kevinj24fr/peek-series) | A deterministic inspection layer for bioinformatics file formats |
| [**karna**](https://github.com/3DBMandNE-Lab/karna) | Out-of-core, auditable compute engine for transcriptomic analysis. |
| [**karnaplot**](https://github.com/kevinj24fr/karnaplot) | The visualization layer for karna |
